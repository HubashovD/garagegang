Створено Текстами

Інфраструктурний капітал є базовим для міста та описує сукупність умов, в яких розгоратається креативний потенціал. Це фізичні об’єкти, а також механізми, що уможливлюють процес обігу і конвертації капіталів. Ключовими тут є такі показники, як: законодавча система, індекс сприйняття корупції, толерантність і свобода самовираження, наявність і якість публічного простору, заклади культури і освіти (взяті передусім як приміщення і матеріально-технічна база), фінансова інфраструктура.

Теми капіталу: Система правосуддя / Боротьба з корупцією / Свобода слова / Свобода самовираження / Інформаційні та комунікаційні технології / Комунальні споруди / Громадські місця / Публічні простори / Заклади культури / Сервісні центри / Центри соціального обслуговування / Екологічні проєкти / Урбаністичні проєкти / Архітектурні проєкти / Рекреаційні споруди / Туризм

Культурний капітал – продукти працівників культурних та креативних індустрій, а також цінності та творчі практики, що змінюють моделі поведінки і дають змогу переосмислювати і дизайнувати життєвий досвід. Наприклад, до таких практик належать читання, відвідання музеїв, концертів, вистав, участь у фестивалях, майстер-класах та інших інтерактивних подіях. Цей вид капіталу також визначає, яким чином у суспільстві відбувається обіг решти капіталів, адже йдеться про систему неформальних правил, які переважно є сильнішими за формальне законодавство.

Теми капіталу: Переклад / Музика / Відео / Анімація / Дизайн / Кіно / Вистава / Перфоманс / Візуальне мистецтво / Арт-об'єкти / Фестивалі / Виставки / Ландшафтний дизайн / Мурали / Хеппенінг / Мистецька акція / Культурна спадщина / Дослідницькі експедиції / Документальні фільми / Ремесла / Гастрономічні та кулінарні витребеньки / Прикраси / Одяг / Культура та освітній туризм

Людський капітал – це знання і навички та їх втілення у різних сферах життя. Цей вид капіталу проявляється через різні форми освіти і самоосвіти, розробки і дослідження (R&D), що здійснюються відповідними лабораторіями чи центрами як навчальними закладами, так і підприємствами.

Теми капіталу: Освіта і навички / Дослідження / Дослідницькі центри / Фаблаби і центри ремісництва / Аналітичні центри / Інноваційні лабораторії / Інкубатори / Стартапи / Креативні хаби / Профорієнтація / Психічне здоров’я / Професійні подорожі / Акселератори підприємництва

Соціальний капітал – довіра, соціальна справедливість, взаємопідтримка участь. Проявляється через програми інклюзії для залучення містян до розв’язання проблем міста та доброчинність.

Теми капіталу: Робота з вразливими групами / Інклюзивнсіть / Толернатність / Права людини / Самовираження / Підвищення довіри / Підтримка інституцій / Спільнодія та кооперація / Культура дарування / Залученість / Підсилення впливу / Інтеграція / Культурний обмін / Розвиток громад / Соціальний бізнес
